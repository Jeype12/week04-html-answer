Html tags use in cozycup boilerplate

  • <!DOCTYPE html>: Declares the document type and version (HTML5).
  • <html>: The root element of the HTML document.
  • <head>: Contains metadata and links to external resources.
  • <meta>: Provides metadata about the document (e.g., character encoding, viewport settings).
  • <link>: Links external resources like CSS files.
  • <title>: Sets the title of the webpage (displayed in the browser tab).
  • <body>: Contains the visible content of the webpage.
  • <header>: Represents the header section of the page.
  • <nav>: Defines the navigation section.
  • <div>: A generic container for grouping content.
  • <img>: Embeds an image in the document.
  • <ul>: Defines an unordered list.
  • <li>: Represents a list item within unordered list or ordered list.
  • <a>: Defines a hyperlink
  • <section>: Represents a thematic grouping of content.
  • <h1>: Defines the main heading (largest heading level).
  • <h2>: Defines a subheading (second-largest heading level).
  • <p>: Defines a paragraph of text.
  • <footer>: Represents the footer section of the page.
